A. 不论设备上是否配置了 BPDU Tunnel 都不会对 MSTP 造成影响  
B. 与用户终端设备相连的端口 MSTP 是否是处于去使能状态或配置为边缘端口  
C. 查看端口配置，确认使能 MSTP 的端口是否使能了协议报文上送命令。如：`bpdu enable`

**正确答案：** A

**解析：**
